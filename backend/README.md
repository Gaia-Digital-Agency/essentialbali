# essentialbali - Backend

This is the backend for the essentialbali application. It is a Node.js application built with Express, Sequelize, and MySQL.

## Getting Started

1.  Install dependencies:
    ```bash
    npm install
    ```
2.  Set up your environment variables by copying `.env.sample` to `.env` and filling in the required values.
3.  Run database migrations:
    ```bash
    npm run db:migrate
    ```
4.  Start the development server:
    ```bash
    npm run dev
    ```

The API will be available at `http://127.0.0.1:8080/api`.
